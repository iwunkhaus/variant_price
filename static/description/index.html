Product/Variant List View With Dynamic Pricelist Price

This module allows to display Multiple Pricelists at the Product/Variant list view dynamically. It allows to edit/update the pricelist price from both Pricelist screen and Product/Variant list view. It works for the pricelist(s) item based on the Product/Variant with fixed price.

Module Features

  • Provides configurable option to Enable the Pricelist(s) dynamically at the Product list view or Product-Variant list view.
  • Provides option to enable the Pricelist(s) in the Product-Variant list view [provided the Variants option checked from Sales -> Configuration -> Settings -> Product Catalog section].
  • Enables Multiple Pricelists [with a fixed price] in the Product/Variant list view [depending on the - Enable Pricelist In - options - 1) Product List View 2) Product Variant List View].
  • Provides option to edit/enter the Pricelist Price from the Product/Variant list view and vice versa.
  • Provides option to edit/enter the Pricelist Price from the Product/Variant form view as well.
  • Module compatible with Single and Multi-Company setup.
  • Module supports company_dependent/Multi-Company Rules. Pricelist Price can be configured company specific, in which those pricelists are allowed to edit/update the price from Product/Variant list view with pricelist company specific login only.
  • Module supports easy Delete of Pricelist(s) which in-turn removes the dynamic pricelists from Product/Variant list view.
  • Module supports easy Archive/UnArchive of Pricelist(s). On Archive it removes the dynamic pricelists from Product/Variant list view and On UnArchive it enables back in the respective Product/Varaint List View [provided if they were enabled during Archive].

Usage

Dynamic Pricelist at Product List View



1. Enable "Product Dynamic Pricelist Access" to show pricelist to that user.



2. Enable "Pricelists" with advance price rules in order to display product pricelists.



3. Now go to pricelist and create a pricelist
if you want to show the pricelist in Product List View, Enable "Product List View" option



4. Click on Add a line to add product to the pricelist
This will open wizard, here you can apply a pricelist on the product(product template) and fixed price only



5. Click on Save



6. Pricelist inside the product list view,
here we can see all the pricelist where Product List View checkbox is ticked



7. you can edit the pricelist value for particular product from list view



8. when you edit the pricelist value from the list view, that price auto-change in the pricelist form.



9. You can add any product to pricelist that is not yet configured/present in the respective pricelist item just by adding pricelist price from the list view
it gets added into the pricelist and price will be updated



10. you can also add product/update Pricelist price to the pricelist from Product form view.



11. you can also create the company dependant pricelist for the product
to create a company dependant pricelist select the company and add product to the pricelist.




12. In pricelist list view you can see the company dependant and company independant pricelist




13. In product list view you can manage the company dependant and company indendant pricelist value.
for company dependant pricelist you can only see/manage the selected company pricelist, other company pricelist value will remain invisible




14. like this you can manage the multiple company dependant/ company independant pricelist


15. You can delete/archive the pricelist, on delete/archive it will remove the pricelist cloumn from the product list view.



16. Here you can see the deleted/archived pricelist are not visible in Product List View.
